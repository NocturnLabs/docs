# OpenCode Updater

A utility for managing and updating OpenCode installations across your system.

## Features

- **Auto-detection**: Finds existing OpenCode installations
- **Version Management**: Check and compare versions
- **Seamless Updates**: One-command upgrade process
- **Rollback Support**: Revert to previous versions if needed

## Installation

```bash
# Using cargo
cargo install opencode-updater

# From source
git clone https://github.com/yumlabs/yumlabs-opencode-updater
cargo install --path .
```

## Usage

```bash
# Check for updates
opencode-updater check

# Update to latest version
opencode-updater update

# Update to specific version
opencode-updater update --version 1.2.3

# Rollback to previous version
opencode-updater rollback
```
