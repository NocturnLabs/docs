# Yum Scaffold

A flexible project scaffolding CLI that generates boilerplate code for various languages and frameworks.

## Features

- **Multiple Languages**: Rust, Go, TypeScript, Python templates
- **Customizable Templates**: Create and share your own templates
- **Interactive Mode**: Guided project setup with prompts
- **Git Integration**: Automatic repository initialization

## Installation

```bash
# Using cargo
cargo install scaffold

# From source
git clone https://github.com/NocturnLabs/scaffold
cargo install --path .
```

## Usage

```bash
# Initialize a new project
scaffold init my-project

# List available templates
scaffold templates list

# Create from specific template
scaffold init my-project --template rust-cli
```

## Available Templates

| Template | Description |
|----------|-------------|
| `rust-cli` | Rust CLI application with clap |
| `rust-lib` | Rust library with docs |
| `go-api` | Go REST API with Gin |
| `ts-node` | TypeScript Node.js project |
