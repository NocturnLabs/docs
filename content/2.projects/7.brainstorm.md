# Brainstorm

A tool for AI-assisted brainstorming and project ideation.

## Features

- **Idea Generation**: Generate project ideas from prompts
- **Expansion**: Elaborate on initial concepts
- **Structured Output**: Organize thoughts into actionable plans
- **Export**: Save brainstorming sessions

## Installation

```bash
# Clone the repository
git clone https://github.com/yumlabs/yumlabs-brainstorm
cd yumlabs-brainstorm
```

## Usage

```bash
# Start a brainstorming session
brainstorm start "Build a productivity app"

# Expand on an idea
brainstorm expand --idea "task management"

# Export session
brainstorm export --format markdown
```
